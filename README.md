# Barbearia Unicorte

Sistema de gerenciamento para a barbearia fictícia Unicortes. Esse sistema é um projeto para a disciplina de Engenharia de Software no período 2024.1.

## Primeiros Passos

Primeiro, execute o servidor de desenvolvimento:

```bash
npm install
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev
```
